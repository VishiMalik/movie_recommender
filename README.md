#  Movie Recommender System

This is a simple content-based movie recommender system built using the [MovieLens 100k dataset](https://grouplens.org/datasets/movielens/100k/). It allows users to select a movie and see the top 10 most similar movies based on user rating patterns.

##  Features

- Item-based collaborative filtering using Pearson correlation
- Interactive Streamlit app UI
- Real data from MovieLens 100k
- Simple filtering based on rating count
- Jupyter Notebook for model development
- '.py' file for deployment via Streamlit
